#  Expense Tracker CLI

A simple command-line expense tracker built with Python.

## Features
- Add expenses with name and amount
- View all expenses with total
- Delete expense by name
- Save expenses to a file
- Load previous expenses on startup
- Input validation and error handling

## How to Run

1. Clone the repo
2. run the script

## What i learned
- Python functions and dictionaries
- File handling operations(read and write)
- Exception handling
- Building CLI application
